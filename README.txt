The main goal of the project is to detect a mask on a human face.
With the application it is possible to start/stop camera recording, image processing
and uploading data to the ms database.

This project is based on:

- LabVIEW 2020 (32-bit)
- Python3 (32-bit)
- Microsoft sql server express
- Microsoft sql management studio
